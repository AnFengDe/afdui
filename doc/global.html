detailControl

the control definition.
Properties:
Name Type Description
label String control label text
id String control id
type String control type:input, select
format String control value format:text,number
readonly Boolean readonly control can not be edit
maxlength Integer the value maxlength
size Integer control width
mask String the control input mask, see jQuery.maskedit document
message String the control input tip message
validator Function the control value validate function
customData Function the control init value function, will be call when create new data
Source:
  • afdui.tabledetail.js, line 103
Example
 example code:
            {
                "label" : "Name",
                "id" : "name",
                "type" : "input",
                "format" : "text",
                "readonly" : false,
                "maxlength" : 32,
                "size" : 32,
                "validator" : function(value) {
                    return (1 <= value.length && value.length <= 32);
                },
                "customData" : function() {
                    return '2400000';
                },
                "message" : "Name length between in 1 - 32"
            }
